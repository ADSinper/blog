## 破解云盘精灵
教程：

公众号恢复：【云盘】，获取破解材料 — 【yunpan.jar】

**本地运行**  

运行环境  jdk8 

运行端口   666 

访问地址： <http://127.0.0.1:666/> 启动命令 

 在文件根目录 cmd执行 java -jar yunpan.jar

运行完之后，稍等一段时间，访问 <http://127.0.0.1:666/>  就可以愉快地使用了。
